# ASCII_ART
Convert images to ASCII characters images
How to build project:

1.You should have Visual Stuido 15 or higher.  
2.Install OpenCV from https://github.com/opencv/opencv.  
3.Build OpenCV for your version of Visual Studio.  
4.Copy 'opencv' folder to folder with this project.  
5.Build project in Visual Studio Ctrl+Shift+B.  
6.Specify command line arguments in Visual Studio and run it or do it from cmd.
